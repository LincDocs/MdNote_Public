# UI库

这里总结一些前端常用的图标库、图片库、svg库。挺多的

## 图标类

比较多，直接搜 `icon` 就出来一大堆。
- 开源的 [lucide](https://lucide.dev/) 也是我比较常用的
- 阿里的 [iconfont](https://www.iconfont.cn/)
- [iconify](https://icon-sets.iconify.design/)
- [fontawesome](https://fontawesome.com/search)
- 像组件库 ant、element 等都有提供
- *(需要注意非开源的，哪怕免费也可能有版权问题)*
- *(使用图标需掌握自行从svg中调整图标大小、颜色、粗细的手段。有的网站能快捷操作，但有的不提供，需要手动调整)*
- svg库：不仅仅是图标，因为上面的图标部分大概率都是有对应的svg的。这里更强调的是背景svg
  - https://www.freepik.com/free-photos-vectors/svg-background
