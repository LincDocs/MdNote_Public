# 如何搭建一个个人博客？

以下是我所了解的一些框架：

[table|exTable|addClass(ab-super-width)]

- < 框架     | 语言  | 描述
- vuepress  | ts/js | 功能强大，自定义能力强
  多文件配置
  ```bash
  略
  ```
- vitepress | ^    | ^
- **LincWorkflow** | ts/js | 基于vuepress hope主题进行修改
  允许零配置即可用，可选 theme.cover.ts、config.cover.ts 覆盖配置
  导航配置默认自动生成，支持 mdbook/hugo 的 SUMMARY.md
  支持使用代理文件声明，分离文档位置和构建位置
  集成了一些 obsidian 用户常用的工具
  ```bash
  # (复制 https://github.com/LincDocs/Workflow 中的 workflows 文件到你的项目即可)
  ```
- Hugo      | go    | 
  可选 hugo.toml 单文件配置，导航为 src/SUMMARY.md
  ```bash
  hugo new site my-hugo-docs
  cd my-hugo-docs
  ```
- gitbook   | ts/js | 
  零配置
  ```bash
  npm install -g gitbook-cli
  gitbook init
  gitbook build # gitbook serve
  ```
- mdbook    | rust | 不支持中文搜索
  可选的 book.toml 单文件配置，导航为 SUMMARY.md
  ```shell
  cargo install mdbook # 工作流可以用 uses: peaceiris/actions-mdbook@v2 安装mdbook
  mdbook build
  ```
- mkdocs    | python | 
  使用 mkdocs.yml 单文件配置，导航为yml里的nav
  ```bash
  pip install mkdocs-material
  cd docs
  mkdocs gh-deploy --force
  ```
- Astro     |        | 不熟
