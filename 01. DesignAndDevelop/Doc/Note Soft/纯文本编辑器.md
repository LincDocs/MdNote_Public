# 纯文本编辑器

## 调研 (2025-09)

要求:

- 可**局部内存编辑**（Hex视图、文本编码视图、二禁制文件、超大文件局部加载）
- 轻便好用
- 仅需要文本编辑，不要求程序/md编辑，但有高亮最好。不要求运行
  - 即不包括更重量级一些的 Sublime、VSCode 等
