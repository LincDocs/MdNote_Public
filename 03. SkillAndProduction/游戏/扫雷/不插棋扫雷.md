# 不插棋扫雷

对于能批量标1+个雷，或批量开1+/2+个雷，标雷再批量开雷有时会比较快

但对于一些简单的收尾，不插棋直接判断往往是更快的。特别是当你熟练后

下面进行一些提速训练 —— 不插棋扫雷

## 对比减法

这个是别人教我的，和在 [扫雷高级定式](扫雷高级定式.md) 中提到的扫雷不等式减法不同，这里更多的是比较

```
# 四共区, 三不共
a a a
b N b
b M b
c c c
```

例如这里中，b是共区。假设M>N，两个数字之差 $M-N$ 就是c比a多出来的雷数。
无论b区存在几个雷，该结论均成立

这种对比可以是两个数字格直接相邻 (4个共区)，也可以是其他情景，如:

```
# 三共区，五不共
a a a
a N a
b b b
c M c
c c c
```

```
# 二共区，五不共
a a a
a N b c
a b M c
  c c c
```

```
# 二共区，六不共
a a a
a N b c c
a a b M c
    c c c
```

一共四种位置关系

### 相邻 - 14、25、36、47 定式

```
a a a    a a a    a a a    a a a
b 1 b    b 2 b    b 3 b    b 4 b
b 4 b    b 5 b    b 6 b    b 7 b
c c c    c c c    c c c    c c c
```

这里由于 $M-N=3$，且非共区数为三，所以a均安全，c均有雷

这里可以发现不插棋所使用的技巧的特征 —— 不优先确定雷的具体位置，而是需要快速确定哪些可以直接开
